GitLab integration

        
      

      
Issue-driven signing loop. When enabled, each new CSR job
         opens a GitLab issue (CSR pasted in, assigned to the signers) in the
         project below; signers paste/attach the signed cert in the issue and
         the dashboard attaches it to the job. Stored in
         /etc/csr-dashboard/integrations.conf.

      

      
      
      
      
      
      

        
        
      

      
      
      
      
      
In GitLab: Project > Settings > Webhooks → URL
         —, Secret token = the value above,
         enable the Comments and Issues events triggers.
